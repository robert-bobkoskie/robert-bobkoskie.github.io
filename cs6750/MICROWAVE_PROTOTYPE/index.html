About the Project

This website is part of a human factors study to improve ergonomics and interface design for the Visually Impaired (VI) population. The concept originates from a CS 6750: Human-Computer Interaction research project.


The results from this project will advance research for developing better interfaces that improve accessibility and user experience for visually impaired audiences. This project will culminate with a paper commensurable for publication.


This Wireframe is a Proof of Concept (POC) design for a novel microwave interface. The primary objective of this interface is to improve user interaction with a microwave for the Visually Impaired (VI) community. This interface will evolve as additional Needfinding is integrated into an agile (iterative) design process. The final product will be a much larger form-factor for the UI that's better suited for smartphone displays.
